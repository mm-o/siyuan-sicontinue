## ✨ 核心特性

- **上下文感知** - 理解你的写作上下文，提供智能建议
- **自动补全** - 基于内容的智能文本续写
- **无缝集成** - 在思源笔记中自然流畅地工作

---

## 🚀 快速开始

### 安装
1. 打开思源笔记 → `设置` → `集市` → `插件`
2. 搜索 "思续" 或 "SiContinue"
3. 点击安装并启用
